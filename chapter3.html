Chapter 3:Random Variables and Probability Distributions

Planned Hours: 07

A random variable is a function that assigns a real number to every outcome of a random experiment.

Example:
Tossing a coin 3 times → Let X = number of heads obtained.

1. Types of Random Variables (CO2 – L1)

Random variables are classified into:

(a) Discrete Random Variable

Takes countable values.

Example Possible values
Number of heads in 3 coin tosses 0, 1, 2, 3
Number of defective items 0, 1, 2, ...

(b) Continuous Random Variable

Takes any value in an interval.

Example Values
Time to complete a job Any real value
Height of students 150.3, 150.35, 150.356…

2. Probability Distribution, CDF & Graph (CO2 – L2)

(a) Probability Distribution of a Discrete Random Variable

If X is discrete, its probability distribution is given by:

P(X = xi) = pi, ∑pi = 1

Example: Tossing a coin twice, X = number of heads.

X 0 1 2
P(X) 1/4 1/2 1/4

(b) Cumulative Distribution Function (CDF)

F(x) = P(X ≤ x)

x F(x)
x < 0 0
0 ≤ x < 1 1/4
1 ≤ x < 2 3/4
x ≥ 2 1

The CDF graph is a step function.

3. Mean and Variance (CO2 – L3)

(a) For Discrete Random Variable

μ = E(X) = ∑ xi pi

σ² = Var(X) = ∑ xi² pi - μ²

(b) For Continuous Random Variable

μ = ∫-∞∞ x f(x) dx

σ² = ∫-∞∞ x² f(x) dx - μ²

4. Application of Probability Distributions (CO2 – L3)

Used in:

Distribution Application
Binomial No. of successes in fixed trials
Poisson No. of arrivals in fixed time
Normal Heights, weights, errors

5. Marginal & Conditional Distributions (CO2 – L3)

Joint Distribution:
P(X = xi, Y = yj)

Marginal Distribution:
P(X = xi) = ∑j P(X = xi, Y = yj)
P(Y = yj) = ∑i P(X = xi, Y = yj)

Conditional Distribution:
P(X = xi | Y = yj) = P(X = xi, Y = yj) / P(Y = yj)
